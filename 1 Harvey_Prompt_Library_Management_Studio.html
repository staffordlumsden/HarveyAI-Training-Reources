Start with purpose, audience, and risk before choosing a prompt.

The Prompt Library is useful when prompts are curated, localised, and governed. Treat existing library prompts as starting points that need review before faculty or student use.

Purpose

This studio helps educators prepare prompt-library entries that are bounded, audience-appropriate, jurisdictionally localised, and reviewable before testing in Harvey.

Expected action

Enter the teaching and governance context, select a prompt category, review the generated prompt, then decide whether it is suitable for faculty-only use, student-facing use, clinic use, or restricted access.

Boundary

This page is a local planning aid. Nothing entered here is transmitted to Harvey. Copy the final prompt or library record into Harvey only after checking the review and governance criteria.

Working principle

Use the Library to support disciplined prompt reuse. A prompt is not ready merely because it exists; it should name the audience, jurisdiction, task, limits, output format, and verification requirement.

Select the prompt category that best matches the library entry.

Choose the smallest useful category. Library prompts should support a defined legal education task rather than invite broad, unsourced, or final-answer outputs.

How to choose

  1. Start with the teaching or workflow need.
  2. Choose the lowest-risk useful prompt category.
  3. Check whether the prompt should be faculty-only or student-facing.
  4. Reject prompts that invite final legal advice, unverified authority use, or complete student answers.

What to avoid

Avoid generic prompts that ask Harvey to “explain the law” without jurisdiction, audience, source limits, output format, or verification instructions. Do not expose faculty-oriented prompts to students unless they have been deliberately adapted.

Selection rule

Use a category that helps users practise or perform a reviewable step: structure, source checking, comparison, drafting for review, or workflow planning.

Localise the prompt before it becomes a reusable library asset.

The suggested prompt combines the selected category with your context. Edit it to remove inappropriate assumptions, especially US legal framing, unsupported authority use, or audience mismatch.

Review sequence

  1. Check that audience, jurisdiction, topic, and intended use are explicit.
  2. Replace generic or US-focused assumptions with local legal education context.
  3. Add source limits, uncertainty instructions, and output format requirements.
  4. Remove wording that asks for final legal advice, complete model answers, or unverified doctrinal claims.

Quality standard

A library-ready prompt should make the output easier to check. It should tell users what the prompt is for, when not to use it, and what must be verified before relying on the output.

Suggested localised prompt
Before testing

Read the prompt as though it were a public instruction to a research assistant. Any ambiguity that would produce poor human work is likely to produce poor tool-supported output.

Decide who should see, use, or change the prompt.

Prompt-library governance is about visibility, contribution rights, and suitability. A prompt may be sound for staff but inappropriate for students without stronger limits and learning instructions.

Visibility decision

Classify the prompt as faculty-only, teaching-team only, student-facing after review, clinic use only, or not ready to share. Do not assume all library users should see the same prompt.

Contribution control

Student contribution to a central prompt library should be restricted unless a formal local process has authorised it. Prompt sharing should be curated rather than open-ended.

Localisation rule

If the prompt originated from a US-focused or generic source, check terminology, jurisdiction, procedure, professional responsibility framing, and authority expectations before use.

Governance warning

If a prompt could help students bypass the intended learning task, expose staff-only criteria, or generate unverified legal claims, keep it restricted and redesign it before student use.

Confirm the prompt is ready to test, save, or restrict.

Use the checklist to decide whether the prompt can be tested in Harvey, saved as a faculty resource, adapted for students, or held back for governance review.

Next action

Copy the final prompt and the library record. Test the prompt in Harvey using low-risk material, revise the wording, and only then save or share the prompt in the relevant Library area.
